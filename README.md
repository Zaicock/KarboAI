# KarboAI

Official Python SDK for KarboAI APIs.

## Installation

```bash
pip install karboai
```

## Usage

```python
from karboai import KarboAI

api = KarboAI()

communities = api.community.all()

print(communities)
```

## Features

- Community APIs
- Blogs APIs
- Clean structure
- Easy to use
- Expandable
